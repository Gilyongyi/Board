개인프로젝트

프로젝트 내용 : 게시판 만들기

요구사항

  - 기본적인 CRUD, 페이징 처리, 검색, 파일업로드, 댓글

기간 : 2016.10.24(월) ~ 2016.11.10(목)

사용기술

  - Java , HTML5, CSS3, JavaScript, jQuery, Handlebars, Bootstrap

사용 Tool

  - Eclipse, Gradle, exERD, MySQL Workbench, Oracle Developer

사용 DBMS

  - Oracle11g, MySQL

특징

  - 회원가입이 필요없는 자유게시판

  - 게시물 비밀번호로 글 수정 / 삭제

  - HTML만을 사용하여 Back-end와 Front-end를 완벽히 나눔.

  - 모든 데이터는 JSON 데이터로 주고 받음. Restful Server로 View 만 바꿔준다면 재사용 가능

  - MySQL ,Oracle 두가지 DBMS 사용가능

  - ajax를 이용한 리로딩 없는 페이지

  - 무한스크롤 페이징 처리
